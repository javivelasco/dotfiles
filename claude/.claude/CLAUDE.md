# Personal Preferences

## Environment
- Shell: fish
- Editor: neovim
- Terminal: ghostty
- Multiplexer: tmux
- Package manager: pnpm (for JS projects)

## Code Style
- Prefer TypeScript over JavaScript
- Use functional patterns where practical
- Keep functions small and focused
- Favor composition over inheritance

## Communication
- Be direct and concise
- Skip unnecessary explanations for simple tasks
- No emojis unless explicitly requested

## Workflow
- Use stow for dotfiles management
- Projects typically live in ~/Code
